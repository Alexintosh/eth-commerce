Features

This is what EthCommerce provides you:

Zero Fees

We generate a P2P transaction between your customer and you, which has zero fees (unlike credit cards or payment gateways)

Almost instant

Transactions will be confirmed in about a minute, which guarantees that the transaction is irreversible and your funds are safe

Easy to integrate

With a couple of lines of javascript you can start accepting Etherum payments on your website. 100% javascript and 0 dependencies!

Mobile App Support

You can use the same library to integrate payments on your mobile React Native app (iOS and android)

Multiple Currencies

We support multiple currencies (USD, EUR, CNY, JPY, etc) and we get a real time rate of the ethereum price in your local currency before initiating the transaction.

dApp Friendly

EthCommerce "just works" with most dApp browsers like (Mist, Cypher, Toshi, Status) and the well known Metamask extension

Get started

EthCommerce is really easy to integrate on your website. This is how you do it:

Pay with Ethereum button

As you can see it's very easy to integrate. You can choose the default "Pay with ethereum" button and pass some minimal configuration like the DOM element that will render the button, that amount to charge, the currency that represents that amount, the address where you want to receive the ETH and two callbacks (success and error)

Custom Button

It's very easy to customize the button by overriding the default styles (.eth-btn, .eth-icon-wrapper, .eth-icon-svg, .eth-btn-text, #eth-btn-text, etc.) You can change the text by using type: CUSTOM and setting the option "label"
